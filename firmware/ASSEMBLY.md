# Инструкция по сборке — VehicleLogger ESP32

## Компоненты

| # | Модуль | Кол-во | Примечание |
|---|--------|--------|------------|
| 1 | ESP-WROOM-32 (DevKit) | 1 | Основной контроллер |
| 2 | WCMCU-230 (SN65HVD230) | 1 | CAN-трансивер, 3.3V |
| 3 | GY-GPSV3-NEO-M8N | 1 | GPS-модуль, UART, 3.3V |
| 4 | MPU-6050 (GY-521) | 1 | Акселерометр + гироскоп, I2C |
| 5 | Макетная плата + провода | 1 | Для прототипа |
| 6 | USB-кабель (micro-USB) | 1 | Питание + прошивка |

---

## Схема подключения

Плата: **LIVE MINI KIT ESP32** (ESP-WROOM-32)

```
              LIVE MINI KIT ESP32
         ┌─────────────────────────────┐
         │  GND TXD     │     RST GND  │
         │  IO27 RXD    │     SVP NC   │
     CAN │  IO25 IO22 ──┼──── IO26 SVN │
     CAN │  IO32 IO21 ──┼──── IO18 ──  │ MPU
         │  TDI  IO17 ──┼──── IO19 ──  │ MPU
     GPS │  IO4  IO16 ──┼──── IO23     │
         │  IO0  GND    │     IO5 TMS  │
         │  IO2  VCC    │     3.3V NC  │
         │  SD1  TDO    │     TCK SD2  │
         │  CLK  SDD    │     SD3 CMD  │
         └──────────────┴──────────────┘
                      USB

  IO22 ← CTX (CAN TX)     IO18 ← SDA (MPU-6050)
  IO21 ← CRX (CAN RX)     IO19 ← SCL (MPU-6050)
  IO16 ← TX  (GPS → ESP)
  IO17 ← RX  (ESP → GPS)

    WCMCU-230 (CAN)         MPU-6050 (I2C)          NEO-M8N (GPS)
    ┌─────────────┐          ┌───────────┐           ┌───────────┐
    │ CTX ── GPIO 22         │ SDA ── GPIO 18        │ TX ── GPIO 16
    │ CRX ── GPIO 21         │ SCL ── GPIO 19        │ RX ── GPIO 17
    │ 3V3 ── 3.3V            │ VCC ── 3.3V           │ VCC ── 3.3V
    │ GND ── GND             │ GND ── GND            │ GND ── GND
    │ CANH ── CAN-шина       └───────────┘           └───────────┘
    │ CANL ── CAN-шина
    └─────────────┘
```

## Распиновка

| ESP32 GPIO | Направление | Модуль | Пин модуля | Сторона платы |
|------------|-------------|--------|------------|---------------|
| **GPIO 22** | OUT | WCMCU-230 | CTX | Левая inner, стр. 3 |
| **GPIO 21** | IN | WCMCU-230 | CRX | Левая inner, стр. 4 |
| **GPIO 16** | IN | NEO-M8N | TX | Левая inner, стр. 6 |
| **GPIO 17** | OUT | NEO-M8N | RX | Левая inner, стр. 5 |
| **GPIO 18** | I/O | MPU-6050 | SDA | Правая inner, стр. 4 |
| **GPIO 19** | I/O | MPU-6050 | SCL | Правая inner, стр. 5 |
| **3.3V** | PWR | Все модули | VCC/3V3 | Правая, стр. 8 |
| **GND** | PWR | Все модули | GND | Левая, стр. 1/7 |

### Важно

- Все модули работают от **3.3V**. Не подавать 5V на NEO-M8N и MPU-6050.
- WCMCU-230 питается от 3.3V (не 5V вариант TJA1050).
- CAN-шина (CANH/CANL) на этапе диагностики не подключается — трансивер тестируется только инициализацией.
- GPS-антенна должна «видеть небо». В помещении фикс может не определиться — это нормально, NMEA-данные всё равно пойдут.

## Программная часть

### Требования

| Инструмент | Версия | Установка |
|------------|--------|-----------|
| PlatformIO CLI | latest | `pip install platformio` |
| Python | 3.8+ | python.org |
| USB-драйвер | CP2102 или CH340 | зависит от DevKit |

> **Альтернатива:** можно использовать расширение PlatformIO для VS Code вместо CLI.

### Определение COM-порта

После подключения ESP32 по USB:

```bash
# Windows — Диспетчер устройств → Порты (COM & LPT)
# Или через PlatformIO:
pio device list
```

### Настройка Wi-Fi

Перед первой прошивкой отредактировать `include/config.h`:

```c
#define WIFI_SSID     "имя_сети"
#define WIFI_PASSWORD "пароль"
```

### Сборка

```bash
cd firmware
pio run
```

Первая сборка скачает тулчейн ESP32 и библиотеки — займёт несколько минут.

### Прошивка

```bash
pio run -t upload
```

Если порт не определяется автоматически, указать вручную:

```bash
pio run -t upload --upload-port COM3
```

### Serial Monitor

```bash
pio device monitor
```

Скорость: 115200 бод (настроена в `platformio.ini`).

Для выхода: `Ctrl+C`.

### Устранение проблем

| Проблема | Что проверить |
|----------|---------------|
| Wi-Fi: FAIL | Правильность SSID/пароля в `config.h`, расстояние до роутера |
| MPU-6050: FAIL | Провода SDA→GPIO21, SCL→GPIO22, питание 3.3V, пайка |
| CAN init: FAIL | Провода CTX→GPIO22, CRX→GPIO21, питание WCMCU-230 |
| GPS UART: 0 байт | Провода TX→GPIO16, RX→GPIO17 (перепутаны?), питание |
| GPS NMEA: FAIL | Байты есть но NMEA не парсится — проверить baud rate (9600) |
| Порт не найден | Установить USB-драйвер (CP2102 / CH340), сменить кабель |
